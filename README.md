# Dokumentation Woche 1

In dieser Woche habe ich die Grundlagen der Webentwicklung vertieft.

### Umgesetzte Punkte:
- **HTML Struktur**: Verwendung von semantischen Elementen wie `header`, `nav`, `main` und `section`.
- **CSS Styling**: Einbindung eines externen Stylesheets zur Trennung von Inhalt und Design.
- **Layout**: Die Navigation wurde mit **Flexbox** realisiert, während die Projekt-Sektion ein **CSS Grid** nutzt.
- **Responsive Design**: Mittels Media Queries passt sich das Layout an mobile Endgeräte an (Wechsel von 3-Spalten auf 1-Spalte).
- **Struktur**: Saubere Dateitrennung und Einhaltung der Benennungskonventionen.
